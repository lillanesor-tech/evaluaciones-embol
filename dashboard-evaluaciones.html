Configuración — Conexión a Google Sheets
Sigue estos pasos una sola vez. Luego el dashboard se actualiza automáticamente cada vez que alguien envía una evaluación.
1
Crea el Google Sheet
En Google Drive, crea una hoja de cálculo nueva. Nómbrala "Evaluaciones Promed Embol". Necesitarás dos pestañas: medico_a_enfermero y enfermero_a_medico.
2
Crea el Apps Script
En el Sheet, ve a Extensiones → Apps Script. Borra todo el contenido y pega este código:
const SHEET_ID = 'ID_DE_TU_GOOGLE_SHEET';

function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const ss = SpreadsheetApp.openById(SHEET_ID);
    const sheetName = data.tipo === 'medico_a_enfermero'
      ? 'medico_a_enfermero' : 'enfermero_a_medico';
    let sheet = ss.getSheetByName(sheetName);
    if (!sheet) sheet = ss.insertSheet(sheetName);

    // Cabeceras si está vacío
    if (sheet.getLastRow() === 0) {
      const headers = Object.keys(data);
      sheet.appendRow(headers);
    }
    sheet.appendRow(Object.values(data));

    return ContentService
      .createTextOutput(JSON.stringify({ok:true}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({ok:false,err:err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  const ss = SpreadsheetApp.openById(SHEET_ID);
  const result = {};
  ['medico_a_enfermero', 'enfermero_a_medico'].forEach(name => {
    const sheet = ss.getSheetByName(name);
    if (!sheet) { result[name] = []; return; }
    const rows = sheet.getDataRange().getValues();
    const headers = rows[0];
    result[name] = rows.slice(1).map(r => {
      const obj = {};
      headers.forEach((h,i) => obj[h] = r[i]);
      return obj;
    });
  });
  return ContentService
    .createTextOutput(JSON.stringify(result))
    .setMimeType(ContentService.MimeType.JSON);
}
3
Despliega como Web App
En Apps Script, haz clic en Implementar → Nueva implementación. Elige tipo "Aplicación web". Configura: Ejecutar como: Yo · Quién puede acceder: Cualquier usuario. Copia la URL que aparece.
4
Pega la URL aquí
Esta URL conecta el dashboard con tu Google Sheet en tiempo real.
También actualiza la variable SCRIPT_URL en los dos formularios HTML con esta misma URL.
5
Carga datos de prueba (demo)
Si aún no tienes evaluaciones reales, carga datos demo para ver el dashboard funcionando.
